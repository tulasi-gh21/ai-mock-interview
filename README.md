# AI Mock Interview System

An AI-powered mock interview platform built using Next.js, Firebase, Gemini AI, and TypeScript.

## Features

- AI-generated interview questions
- ATS Resume Score Analysis
- Missing Skills Detection
- AI Feedback Generation
- Interview Analytics Dashboard
- PDF Interview Report Export
- Authentication System
- Dark Mode
- Interview History Tracking
- Responsive UI

## Technologies Used

- Next.js
- TypeScript
- Firebase
- Gemini AI API
- Tailwind CSS
- jsPDF
- Recharts
